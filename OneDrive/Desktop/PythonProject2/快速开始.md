## 🎯 9. 性能优化建议

如果数据量增大导致运行缓慢：

1. **减少特征维度**
   ```python
   extract_features(X_train, X_val, X_test, max_features=1000)
   ```

2. **使用更简单的模型**
   - 朴素贝叶斯最快
   - 逻辑回归次之
   - SVM 最慢

3. **并行计算**
   ```python
   # 在 GridSearchCV 中设置 n_jobs=-1
   GridSearchCV(..., n_jobs=-1)
   ```
